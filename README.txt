SDA2 - TP1 - NATHAN SALEZ
-------------------------

Pour compiler le projet, taper make dans le répertoire courant.

3 exécutables sont générées lors de la compilation :
- mainTestLib.exe = test de la bibliothèque libPile. (partie 1)
- mainChaine.exe = génération des permutations d'un mot. (partie 2)
- mainDames.exe = génération de solutions, en notation échiquéenne ou en affichage pseudo-graphique, au problème des huits dames. (partie 3)

Pour accéder à la documentation du projet, exécuter le fichier doc/doxygen/html/index.html sur un navigateur web.

Pour obtenir les temps de recherche en fonction de la taille de l'échiquier pour le problème des dames, ouvrir le fichier 'Compte Rendu.ods'.

Les images prouvant les tests sont accessibles dans le répertoire imageTest.
